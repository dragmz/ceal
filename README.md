## Usage

```cpp
#include <avm.hpp>

void init()
{
	// initialize app
}

void noop()
{
	// insert NoOp logic here
}

uint64 avm_main()
{
	if (avm_txn.ApplicationID == 0)
	{
		init();
		return 1;
	}

	if (avm_txn.OnCompletion != NoOp)
	{
		avm_err();
	}

	noop();
}
```

```bash
go run cmd/ceal/main -path contract.cpp
```
